# e-plantShopping

## Overview
Paradise Nursery Shopping Application is a React + Redux-based web app designed for browsing and purchasing houseplants online. It provides a simple shopping cart experience where users can explore plant categories, view details, and manage their cart items dynamically.

## Features
Landing Page: Welcome message and navigation to product listings.
Product Listing: Plant cards with images, descriptions, prices, and "Add to Cart" buttons.
Shopping Cart: Displays selected items, allows quantity adjustments, and calculates totals.
State Management: Redux Toolkit integration for cart actions (add, remove, update).
Deployment: Hosted on GitHub Pages for easy access.

## Technologies Used
- React (functional components, hooks)
- Redux Toolkit (state management)
- Vite (build tool)
- GitHub Pages (deployment)
